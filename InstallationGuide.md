# Jenkins Installation Guide

## Requirements
- Java JDK 21
- Git for Windows
- Apache Maven
- Jenkins LTS

## Installation Steps
1. Install Java and set JAVA_HOME
2. Install Git and Maven
3. Install Jenkins using MSI installer
4. Unlock Jenkins and install recommended plugins
5. Configure Global Tools (JDK, Git, Maven)
6. Set Jenkins URL

## Verification
- Jenkins dashboard loads at http://localhost:8080
- Tools visible in Global Tool Configuration
