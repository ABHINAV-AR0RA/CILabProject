# Jenkins User Manual

## Multibranch Pipeline Job
- Automatically discovers branches
- Runs pipeline defined in Jenkinsfile
- main → build + test
- feature → test only
- release → release checks

## Freestyle Job
- Pulls code from GitHub
- Runs Maven build
- Publishes test results
- Archives artifacts

## Viewing Results
- Dashboard shows job status
- Console output shows build logs
- Artifacts available after build
